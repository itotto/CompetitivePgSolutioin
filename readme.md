# プロジェクト名

このプロジェクトは AtCoder の問題を解いた履歴管理するためのプロジェクトです  

## サイト

https://atcoder.jp/home


# 構造

contests フォルダーには Beginner Contest の回答を日付ごとに分けて格納しています  
heuristiccontests フォルダーには Heuristic Contest の回答を管理しています  

books フォルダーには書籍と連動した練習問題の解答を格納しています  


## サイト

https://atcoder.jp/contests/math-and-algorithm
https://atcoder.jp/contests/tessoku-book


## 使い方

このソリューションのビルドは Windows11 + Visual Studio 2022 で行っています。
